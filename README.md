# Good Faith Calculator
## About
**Good Faith Calculator** (also packaged as **Estimator**) is a desktop Java application for building **good-faith cost estimates** in a clinical or practice setting. Staff enter patient and addressee details, pick services from a fee schedule, apply insurance-specific contracted rates, and generate a printable **Word document** from templates.
The app is aimed at teams that need repeatable estimates with:
- Multiple **estimate tabs** (separate scenarios on one patient)
- A **fee database** (`FeeDB.csv`) keyed by CPT/service code and insurer
- **Contact lookup** from a pipe-delimited phone book
- **Word merge** via [Spire.Doc Free](https://www.e-iceblue.com/Introduce/free-doc-component.html) (`{{placeholder}}` syntax)
> **Note:** This tool supports internal estimation workflows. It is not legal or billing advice; verify outputs against payer rules and your compliance policies.
